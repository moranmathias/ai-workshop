# AI Workflow

## Requirements

- Python 2.7 o greater

## Setup

1. [Install conda](https://conda.io/miniconda.html)
2. Create virtual environment

> `conda env create -f environment.yml`

3. Start enviroment

> `source activate tf`

4. Start jupyter notebook

> `jupyter notebook`

5. Open the link displayed on notebook start log
